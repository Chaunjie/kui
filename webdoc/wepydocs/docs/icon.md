## Icon 图标

### 使用指南
在页面中引入组件
```javascript
import Icon from 'kai-ui/Icon'

components = {
  icon: Icon
}
```

### 代码演示

#### 初始化

```html
<template>
  <icon :type="item" :event="index % 2 ? 'icon-tap' : ''"></icon>
</template>
```

#### 回调
```javascript
events = {
  'icon-tap': function () {
    console.log('点击事件回调')
  }
}
```

### API

| 参数 | 说明 | 类型 | 可选值 | 默认值 |
|-----------|-----------|-----------|-----------|-------------|
| type | 图标的kai-iconfont(例如: 类名为kai-icon-plus type就为plus) | `String` | —— | ` ` |
| detail | 事件回调名 | `String` | —— | ` ` |

### Event

| 事件名 | 说明 |
|-----------|-----------|
| icon-tap| 点击图片事件回调 无参 |
